Quick start

Pick a workflow on the left, or create one from a preset: Home-tasks (merge → generalist → fine-tunes) or Master RGB-D / UMI (record once → extract subsets → two-stage pre-train). Before adding an external dataset, preflight it in Compare datasets; the full recipe lives in the Guide tab and GUIDE.md.

Run and Run all execute stages directly on this machine — no copy-pasting. Every stage is also written to workflows/<name>/scripts/*.sh when you hit Generate scripts, so anything can be re-run by hand or on another box. Watch progress on the Log tab — it lights up while a stage is running.
